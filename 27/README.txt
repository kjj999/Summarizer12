To run the project:

Enter the following command - "java -jar relevanceCalculator.jar 'URL/File_Path' 'Summarizer Number (1/2/3/4/5)'"

There are some dependencies to be installed.

1) NLTK Wordnet : "sudo apt-get install python-nltk"
2) OTS Summmarizer : "sudo apt-get install ots"
3) pip : "sudo apt-get install pip"
4) BS4 : "sudo pip install beautifulsoup4"
5) sumy : "sudo pip install sumy"
NOTE : In case of an error installing sumy, refer to its documentation -> "https://pypi.python.org/pypi/sumy/0.3.0"
